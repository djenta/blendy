# Blendy 2.0.1

This update gives the conversation substantially more room and repairs reference-photo handling.

## What changed

- The titlebar now keeps one short, static description instead of repeating live generation stages.
- Connection status and Stop are compact floating controls instead of a full-width banner.
- The repeated Current checkpoint summary has been removed.
- Check, Stuck, and Where now sit beside Add reference in a compact composer toolbar.
- The 380 x 520 minimum window keeps the conversation readable without horizontal overflow.
- Add reference remains available while the current answer is running, ready for the next message.
- Windows photos with missing or unusual MIME labels are recognized by file extension.
- Selected photos are decoded, resized when needed, stripped of problematic metadata, and re-encoded before LM Studio receives them.
- PNG, JPG, JPEG, JFIF, WebP, BMP, GIF, AVIF, HEIC, HEIF, TIFF, and TIF files are offered by the picker. Formats Chromium cannot decode receive a clear local error instead of an LM Studio HTTP 400.
- The backend rejects malformed image bytes before they reach LM Studio.

## Install

Run `Blendy Local AI Tutor Setup 2.0.1.exe`. The installer updates the desktop app and its managed Blender add-on.

The installer is unsigned, so Windows may show an unknown-publisher warning. `SHA256SUMS.txt` is included with the release for verification.
