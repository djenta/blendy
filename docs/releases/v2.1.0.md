# Blendy 2.1.0

This release rebuilds Blendy's tutoring brain around fresh, authoritative Blender evidence instead of a mandatory model-routing pass.

## What changed

- Ordinary tutoring now goes directly to one grounded answer. Read-only tools are offered only for explicit documentation, workflow, version-sensitive, or approved web needs.
- Every send and regeneration requires a fresh full Blender-window overview. The overview is ordered before the focused editor crop and uses a readable 1440-pixel minimum edge.
- Exact Blender mode, selection, active tool, snapping, pivot, proportional editing, modifier targets, nearby objects, parents, constraints, and scene changes are separated from visual inference.
- The current task is placed last in the prompt, after durable project memory and live evidence. Failed, cancelled, rejected, and incomplete assistant drafts no longer poison later turns.
- Gemma sampling is task-specific: deterministic memory and repair, conservative exact/troubleshooting guidance, normal tutoring, and a separate creative profile. Advanced overrides still win.
- A narrow final-answer check catches contradictions with live Blender mode and performs one tool-free repair pass. Unsafe drafts are withheld rather than silently word-swapped.
- Context compaction no longer deletes the visible transcript. The context menu distinguishes configured limit, loaded model limit, answer reserve, effective input, estimated next request, measured previous usage, and summarized versus stored chat.
- Named reference images stay attached for later steps and regeneration until removed. Prompt captions and receipts distinguish live Blender images from user targets.
- The Project Notebook autosaves, is included before send/regenerate, and uses one consistent 8,000-character limit.
- Chat JSON writes use atomic replacement and backup recovery. Stale temporary screenshot files from a previous crash are cleaned on add-on startup.
- The compact frontend adds a small evidence strip and clearer context details without turning the tutor into a dashboard.

## Privacy

Blendy screenshots are temporary. Capture files are deleted after encoding, stale Blendy capture files are scavenged later, and screenshot/reference pixels are omitted from chat history and diagnostic prompt packets.

Run `Blendy Local AI Tutor Setup 2.1.0.exe`. The installer updates the desktop app and its managed Blender add-on.
